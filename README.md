# Portafolio de Alexander Mañana Sánchez

Portafolio profesional moderno para Especialista en Soporte Técnico Nivel 2 y Desarrollo.

## Estructura del Proyecto

```
portafolio/
│
├── index.html              # Archivo HTML principal
│
├── css/                    # Estilos organizados
│   ├── base/              # Estilos base y configuración
│   │   ├── variables.css  # Variables CSS (colores, fuentes)
│   │   ├── reset.css      # Reset y estilos globales
│   │   └── animations.css # Animaciones keyframes
│   │
│   ├── layout/            # Estructura de la página
│   │   ├── navigation.css # Barra de navegación
│   │   └── footer.css     # Pie de página
│   │
│   └── components/        # Componentes específicos
│       ├── hero.css       # Sección hero
│       ├── about.css      # Sección sobre mí
│       ├── skills.css     # Sección habilidades
│       ├── certifications.css # Sección certificaciones
│       ├── experience.css # Sección experiencia
│       ├── projects.css   # Sección proyectos
│       ├── contact.css    # Sección contacto
│       ├── buttons.css    # Estilos de botones
│       └── cards.css      # Estilos de tarjetas
│
├── js/                    # JavaScript
│   └── main.js           # Scripts principales
│
└── assets/               # Recursos (imágenes, iconos)
```

## Características

- **Diseño Responsive**: Adaptable a todos los dispositivos
- **Animaciones Suaves**: Transiciones y efectos visuales profesionales
- **Código Modular**: CSS organizado por componentes
- **Performance Optimizado**: Carga rápida y eficiente
- **Accesibilidad**: Navegación clara y semántica

## Tecnologías Utilizadas

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- JavaScript Vanilla (ES6+)
- Google Fonts (JetBrains Mono, DM Sans)

## Personalización

### Colores

Edita las variables en `css/base/variables.css`:

```css
:root {
    --primary: #00D9FF;
    --secondary: #0A1F44;
    --accent: #FF3366;
    --text-light: #E8F4F8;
    --bg-dark: #0F172A;
    --bg-card: #1E293B;
}
```

### Contenido

Modifica el contenido en `index.html` para personalizar:
- Información personal
- Habilidades
- Certificaciones
- Experiencia laboral
- Proyectos
- Datos de contacto

## Cómo Usar

1. Descarga o clona el repositorio
2. Abre `index.html` en tu navegador
3. Personaliza el contenido según tus necesidades
4. Sube los archivos a tu servidor web o hosting
